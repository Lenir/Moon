

한국에 사는 뉴비 프로그래머입니다! 1991년생이고, 남성입니다.

[부산대학교][1]에서 컴퓨터 공학을 전공하였습니다. 특별히 한 쪽에 치우치기보다 여러 경험을 하는 것을 좋아합니다.


##### 기타 등등

- 좋아하는 개발 언어는 Python > JS > Java > C, C++ etc. 파이썬은 사랑입니다.
- 주 개발 환경은 mac/OSX이고 주로 사용하는 에디터는 [Atom][2], IDE는 [이클립스][3]입니다.
